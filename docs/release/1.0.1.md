# Lost Cities: Road Fixes 1.0.1

## Requirements

- Minecraft `1.21.1`
- NeoForge `21.1.235` or newer in the 21.1 line
- Lost Cities `1.21-8.3.10` exactly

## What changed

This bug-fix release restores driveable interchange generation when crossing
Lost Cities highways differ by several city levels. Selection now measures the
actual usable turning-route run, converging ramps share a level merge tangent,
and ramp lanes cannot be overwritten by a nearby upper deck foundation.

The reported level-0/level-3 crossing at chunk `(-80, 104)` was regenerated in
the client's exact pack. Its Y=71 and Y=89 decks are connected by a block-level
driveability search. Full evidence is in the
[regression report](../regressions/2026-07-stacked-interchange-level-gap.md).

## Installation

1. Back up the world.
2. Replace version 1.0.0 with `lostcitiesroadfixes-1.0.1+mc1.21.1.jar` on the
   server and every connecting client.
3. Keep Lost Cities `1.21-8.3.10` installed.
4. Generate fresh chunks to see corrected interchange geometry. Existing
   chunks are intentionally not rewritten.
