# Lost Cities: Road Fixes 1.1.1

## Requirements

- Minecraft `1.21.1`
- NeoForge `21.1.235` or newer in the 21.1 line
- Lost Cities `1.21-8.3.10` exactly

## What changed

This fix release makes stack ramps monotonic, widens built-in ramp lanes to 10
blocks, grade-locks every fork and merge, and combines matching stack branches
into one collector before they separate. Full-cloverleaf loops now stay in
their own quadrants inside independent outer ramps.

Interchanges now take precedence over incompatible native intersections. Road
terrain is carved as a taller 8-to-12-block arch, removing low floating
obstructions while protecting planned upper decks.

Implementation and test evidence are recorded in the
[regression report](../regressions/2026-07-monotonic-shared-ramps-and-tunnels.md).

## Installation

1. Back up the world.
2. Replace version 1.1.0 with `lostcitiesroadfixes-1.1.1+mc1.21.1.jar` on the
   server and every connecting client.
3. Keep Lost Cities `1.21-8.3.10` installed.
4. Generate fresh chunks. Existing roads and interchanges are not rewritten.

## Release files

Checksums are generated from the final clean build and recorded before release.
