# Lost Cities: Road Fixes 1.1.0

## Requirements

- Minecraft `1.21.1`
- NeoForge `21.1.235` or newer in the 21.1 line
- Lost Cities `1.21-8.3.10` exactly

## What changed

This release rebuilds built-in ramp terminals around continuous auxiliary lanes
and gradual tapers. A full cloverleaf now has four separate loops and four
separate outer ramps; stack connectors use clear central tiers. Long four-way
crossings deterministically vary across all six applicable families instead of
repeatedly selecting cloverleaf.

Lost Cities buildings are now rejected before placement whenever the complete
single- or multi-chunk footprint, plus one chunk of clearance, would touch a
final road or interchange surface.

The geometry rules are based on FHWA, WSDOT, Caltrans, and MUTCD guidance as
translated in the [research note](../research/2026-07-professional-interchange-geometry.md).
Implementation and exact-pack evidence are in the
[regression report](../regressions/2026-07-professional-cloverleaf-and-building-clearance.md).

## Installation

1. Back up the world.
2. Replace version 1.0.1 with `lostcitiesroadfixes-1.1.0+mc1.21.1.jar` on the
   server and every connecting client.
3. Keep Lost Cities `1.21-8.3.10` installed.
4. Generate fresh chunks. Existing roads, interchanges, and buildings are not
   rewritten.
